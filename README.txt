This is Spin version 6.4.7 with some additional patches from the developer Gerard Holzmann. He said that they will be included in version 6.4.8, but until then they will live here.

You may need to install YACC:
    sudo apt-get install bison

To compile:
    make

To install (it will be installed to /usr/local/bin):
    sudo make install

To clean:
    make clean
